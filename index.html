Windows 11 Installation – USB Bootable Method

🔴 Red Alert – Data & Power Safety

  • Back up every important file from the target PC. This process can erase the entire disk.
  • Use only official Microsoft media creation tools or ISOs.
  • Keep the laptop on AC power during installation. Do not rely only on battery.
  • Unplug other external drives to avoid wiping the wrong disk.

Toolkit / Requirements

  • Another working PC with internet access.
  • 8 GB or larger USB 2.0/3.0 pendrive (all data on it will be erased).
  • Official Windows 11 ISO or Microsoft Media Creation Tool.
  • Target PC that supports Windows 11 (UEFI, TPM 2.0, Secure Boot recommended).

Step‑by‑Step Execution

  1. Download the official Windows 11 installer from the Microsoft website on the working PC.
  2. Insert the USB drive and run the Media Creation Tool or your ISO‑to‑USB tool of choice.
  3. Select your USB drive carefully and confirm that you are okay with erasing its contents.
  4. Create the bootable USB and wait until the tool completes without errors, then safely eject it.
  5. Connect the bootable USB to the target PC and power it on.
  6. Enter the boot menu or BIOS/UEFI (typically by tapping F2, F10, F12, or Del immediately at power‑on).
  7. Choose the USB drive as the first boot device, ensuring UEFI mode and Secure Boot are enabled where supported.
  8. Start the Windows setup, select your language and keyboard layout, and click “Install now”.
  9. Pick “Custom: Install Windows only” for a clean install, then select the correct system drive or partition.
  10. Format only the partition you intend to use for Windows; double‑check drive letters and sizes before proceeding.
  11. Wait for copying and installing to finish; the PC will reboot several times – do not power off.
  12. Remove the USB when Windows first restarts to avoid booting the installer again.
  13. Follow the on‑screen steps to set up region, account, privacy, and network settings.

Verification

  • System boots directly into Windows 11 without showing the installer screen.
  • Settings → System → About reports “Windows 11” and the expected edition.
  • Disk layout shows your system drive with healthy partitions and expected free space.

Pro Tip

After installation, run Windows Update fully once, then create a system restore point. This gives you a clean rollback snapshot before you start installing drivers and random desktop apps.

Linux Setup – Ubuntu (Desktop) Focus

🔴 Red Alert – Partition Safety

  • Never delete or format a partition unless you are 100% sure what it contains.
  • Take a screenshot or note down your current Windows partition layout before starting.
  • Avoid forced power‑off while Ubuntu is resizing or creating partitions.

Toolkit / Requirements

  • Ubuntu Desktop ISO (preferably latest LTS version).
  • 8 GB+ USB drive.
  • Tool like Rufus, balenaEtcher, or Ubuntu Startup Disk Creator.
  • Stable power and internet (for updates and drivers).

Step‑by‑Step Execution

  1. Download the Ubuntu Desktop ISO from the official Ubuntu website.
  2. Create a bootable USB using Rufus or a similar tool, selecting “GPT / UEFI” where available.
  3. Backup important data from the target machine (especially if dual‑booting with Windows).
  4. Boot from the USB by selecting it from the UEFI boot menu.
  5. Choose “Try Ubuntu” first to verify Wi‑Fi, display, and basic hardware support.
  6. Launch the installer from the desktop and select your language and keyboard layout.
  7. Select “Install third‑party software” if you need proprietary codecs or drivers.
  8. Decide between:
    • Erase disk and install Ubuntu – clean Linux‑only system.
    • Install Ubuntu alongside Windows – guided dual‑boot with automatic partitioning.
  9. Review the partition summary carefully before confirming changes.
  10. Set your timezone, username, and strong password when prompted.
  11. Wait for installation to complete; then restart and remove the USB when asked.
  12. Verify that the GRUB boot menu appears and that you can boot into both Ubuntu and Windows (if dual‑booting).

Verification

  • GRUB menu lists Ubuntu (and Windows, if configured).
  • Wi‑Fi, sound, and display run correctly after first boot.
  • Settings → About in Ubuntu shows the expected LTS version and disk usage.

Pro Tip

Right after the first boot, run updates and install only the tools you actually use (browser, editor, terminal tools). A lean Linux install boots faster and survives longer without drama.

EEE Basics – Safely Test a Household Socket with a Multimeter

🔴 Red Alert – Live Mains Voltage

  • You are dealing with 230 V AC. A mistake can be fatal.
  • Do not attempt this if you are wet, barefoot on a damp floor, or unsure about the steps.
  • Use only a multimeter rated for at least 600 V CAT II with intact probes.
  • Keep one hand behind your back when probing live sockets to reduce shock path risk.

Toolkit / Requirements

  • Digital multimeter with AC voltage range (600 V or similar).
  • Probes with undamaged insulation and proper finger guards.
  • Dry footwear and a dry, non‑conductive floor (avoid metal or wet surfaces).

Step‑by‑Step Execution

  1. Inspect the multimeter body and probes for cracks, exposed metal, or melted insulation.
  2. Set the multimeter to the highest AC voltage range (“V~” or “ACV”), typically 600 V.
  3. Insert the black probe into the COM socket and the red probe into the V/Ω socket.
  4. Stand on a dry, insulated surface and keep your fingers behind the probe guards.
  5. Insert the black probe carefully into the left slot (neutral) of the socket.
  6. Insert the red probe into the right slot (live) without letting probes touch each other.
  7. Read the voltage: in India it should be around 220–240 V AC.
  8. Move the black probe to the earth pin (top round hole) and keep the red probe in live.
  9. Confirm a similar 220–240 V AC reading – this checks live‑to‑earth integrity.
  10. Shift the red probe to neutral while black is on earth and verify a small or near‑zero reading.
  11. Remove the probes in reverse order and turn off the multimeter.

Verification

  • Live‑to‑neutral and live‑to‑earth both show ~230 V AC.
  • Neutral‑to‑earth shows only a few volts (ideally < 5 V).
  • Socket does not spark, smell, or heat abnormally during the test.

Pro Tip

Mark sockets that fail these tests with a visible label like “DO NOT USE – FAULTY” and get a qualified electrician to fix them. A 5‑minute test is cheaper than replacing burnt appliances later.

Certificate Fix – Apply for Duplicate JNTUA CMM/PC

🔴 Red Alert – Official Rules Only

  • Use only official JNTUA formats and fee details – rules change; cross‑check once on the university website.
  • Keep photocopies or scans of any remaining marks memos and ID cards.
  • Do not hand over originals to unauthorized agents; work through your college or university exam cell.

Toolkit / Requirements

  • Hall‑ticket number and registered name as per university records.
  • Government ID proof (Aadhaar, PAN, Passport, etc.).
  • FIR copy or loss report (if certificates are lost/stolen), as required by JNTUA.
  • Passport‑size photographs (as per current instructions).
  • Demand draft / online payment proof as per latest JNTUA fee structure.

Step‑by‑Step Execution

  1. Visit the official JNTUA website and navigate to the Examinations or Notifications section.
  2. Locate the latest notification or guidelines for issuing duplicate CMM/PC.
  3. Download the prescribed application form for duplicate certificates.
  4. Fill the form clearly in BLOCK LETTERS with accurate hall‑ticket number, course, year, and contact details.
  5. Attach photocopies of any available old certificates, ID proof, and FIR/loss report if specified.
  6. Prepare the demand draft or online payment as per the exact amount, in favour of the university account mentioned.
  7. Verify that the DD details (name, amount, bank, date) match what is written in the application form.
  8. Route the application through your college principal or exam branch if the notification demands college attestation.
  9. Submit the complete set (application, enclosures, DD/payment proof) at the JNTUA exam branch or by the instructed postal address.
  10. Collect the acknowledgment or inward number for tracking purposes.
  11. Follow up after the indicated processing time via college exam cell or official contact numbers/emails.

Verification

  • Duplicate CMM/PC carries the correct spelling of your name, hall‑ticket number, and course details.
  • Issue date and duplicate note are clearly printed as per JNTUA format.
  • Seals and signatures match other official JNTUA documents.

Pro Tip

As soon as you receive the duplicate certificates, scan and store them in at least two cloud locations plus one offline USB. Future applications (higher studies, jobs, visas) often accept attested photocopies, so keep multiple sets ready.
